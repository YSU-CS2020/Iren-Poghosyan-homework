# Assignment 3: Decision Trees & Ensemble Methods
**YSU CS2020 — Machine Learning**

## Overview
From-scratch implementation of a CART Decision Tree and Random Forest classifier,
applied to the **Wine Quality (UCI)** dataset with three full experiments.

## File Structure
```
code/
├── decision_tree.py   # DecisionTreeClassifier from scratch
├── random_forest.py   # RandomForestClassifier from scratch
├── experiments.py     # All 3 experiments + additional analyses
├── requirements.txt
└── README.md
figures/               # Auto-generated by experiments.py
└── *.png
```

## Setup & Run

```bash
pip install -r requirements.txt
cd code
python experiments.py
```

Figures are saved to `../figures/`.  
The Wine Quality CSV is fetched automatically from UCI.  
If network is unavailable, a synthetic stand-in is used for testing.
